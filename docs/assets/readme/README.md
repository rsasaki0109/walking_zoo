# README Assets

The robot GIFs in this directory are generated by
`tools/render_readme_gifs.py`.

They are deterministic documentation assets based on lightweight gait, footstep,
and runtime-state simulations:

- `simulated_biped_runtime.gif`
- `simulated_estop_stop.gif`
- `simulated_quadruped_trot.gif`
- `simulated_footstep_plan.gif`

These are not part of the walking_zoo runtime and do not turn the project into a
simulator. They exist so the README shows walking behavior instead of static
architecture diagrams or hand-animated toy poses.
